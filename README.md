# ChatooriZubaan 

This project is a restaurant web application called Chatoorizubaan, created for learning purposes.
It is built using the MERN stack (MongoDB, Express, React, Node.js), MySQL, Bootstrap, and EJS templating for the frontend.

The project combines basic frontend and backend in the same folders, there is no separate folder structure for frontend and backend.

## Features

- Display restaurant menu with images and details  
- Dynamic menu rendering using EJS templates 
- Fetch and display orders or customer info from MySQL
- Responsive layout using Bootstrap
- Simple backend routes with Express 
- Basic CRUD operations for learning purposes (Create, Read, Update, Delete)

## Tech Stack

- Frontend: HTML, CSS, Bootstrap, EJS
- Backend: Node.js, Express.js
- Database: MySQL, MongoDB
- Other: JavaScript

## How to Run

!. Clone the project:

```bash
git clone https://github.com/JVKE-KING/chatoorizubaan.git
```

2. Install dependencies:

```
npm install
```
3. Run the server:

```
nodemon app
```

4. Open your browser and go to:

```
http://localhost:8080
```

   
## Notes

- This is a learning project combining frontend and backend in one place.
- EJS is used to render HTML dynamically from server data.
- The project is for educational purposes, to practice MERN + MySQL + Bootstrap integration.
